## Setup
1. Clone the repo
```bash
git clone https://github.com/SwagDemons69/COS301MP`
```
2. Run the dev container in VSCode (`Ctrl+P > Start Dev Container`) or install the dependencies manually:
- Node 16
- Java
- Firebase CLI

2. Install dependencies
```bash
yarn install
```
3. Run the project:
```bash
yarn start:api:dev
yarn start:emulators
yarn start:app:dev
```
